Lab 3: Find the variables

A program contains the following code to calculate the circumference of a bicycle wheel, using the wheel size (diameter):

BEGIN
CONSTANT Pi = 3.14
INPUT WheelSize
Circumference = Pi * WheelSize
OUTPUT Circumference
END

  1. The code uses one constant and two variables.
    a. State the names of the constant and the variables.

    b. Explain one difference between a constant and a variable.

  2. The data type of WheelSize is integer and the data type of Circumference is real number. Explain the difference between an integer and a real number.
